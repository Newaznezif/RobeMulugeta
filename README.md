# Robe Mulugeta | Technology & Education Portfolio

This is the official portfolio platform for **Robe Mulugeta**, a Technology & Education Leader dedicated to building Africa's digital education future.

## 🚀 Key Features
- **3D Network Hero**: Interactive Three.js background.
- **Hybrid AI Assistant**: Conversational agent with [Free Mode] fallback.
- **Impact Showcase**: Animated metrics and project breakdowns.
- **Thought Leadership**: Blog and speaking engagement highlights.

## 🛠️ Tech Stack
- **Next.js 16** (App Router)
- **Tailwind CSS 4**
- **Framer Motion**
- **Three.js**
- **OpenAI API**

## 🏃 Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Setup environment variables**:
   Create a `.env.local` file and add your OpenAI key:
   ```env
   OPENAI_API_KEY=your_key_here
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **View the site**:
   Open [http://localhost:3000](http://localhost:3000)
